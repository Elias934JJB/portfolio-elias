Stage 2 - E-commerce Ballons de Football

Stage réalisé en première année de BTS SIO, centré sur la création d'un site vitrine avec panier pour vendre des ballons de football. L'objectif était de permettre à un client de gérer facilement son catalogue et de proposer une expérience d'achat simple et efficace.

Étapes du projet

  1. Réunion de cadrage avec le client pour identifier les besoins du site
  2. Création de la maquette du site avec Figma
  3. Développement du site avec HTML, CSS et un peu de JavaScript
  4. Intégration d'un panier simple (ajouter/supprimer des articles)
  5. Mise en ligne locale sur un serveur XAMPP pour présentation au client

Compétences développées

  • Conception et maquettage d'interface utilisateur
  • Développement front-end (HTML, CSS, JavaScript)
  • Gestion de projet web
  • Communication client et gestion des besoins
  • Configuration serveur local (XAMPP)

Technologies utilisées

  • HTML5 et CSS3 pour la structure et le style
  • JavaScript pour les interactions utilisateur
  • Figma pour la conception des maquettes
  • XAMPP pour l'environnement de développement
  • Git pour le versioning du code
